# برنامج المحاسبة والمستودعات

برنامج محاسبة وإدارة مستودعات للشركات — يعمل على Windows بدون إنترنت.

## المميزات
- فواتير بيع وشراء مع طباعة احترافية
- إدارة مواد ومخزون
- دعم 3 عملات: دولار / ليرة قديمة / ليرة جديدة
- إدارة زبائن وموردين
- يعمل بدون إنترنت — بيانات محفوظة محلياً

## تحميل البرنامج
👉 [Releases](../../releases) — حمّل آخر إصدار من هنا

## تثبيت للمطورين
```bash
npm install
npm start
```
